# AI Earnings Tracker - 财报追踪器 v2.0

自动追踪**美股、港股、A股**三大市场财报，支持**行业板块筛选**，AI生成结构化摘要。

## 功能特性

- 📅 **财报日历**：每周自动扫描下周财报安排
- 🌍 **三大市场**：美股 🇺🇸 | 港股 🇭🇰 | A股 🇨🇳
- 🏷️ **板块筛选**：科技、金融、消费、医疗等9大行业
- 🤖 **AI摘要**：财报发布后自动生成关键指标分析
- 📱 **飞书推送**：重要信息直达手机
- 🧠 **记忆偏好**：记住用户关注的公司列表

## 使用方法

### 场景1：查看支持的股市

```
用户：有哪些股市？
→ 返回美股、港股、A股列表，含时区和货币
```

### 场景2：查看行业板块

```
用户：美股有哪些板块？
→ 返回美股板块列表：科技、金融、医疗健康等

用户：科技板块覆盖哪些市场？
→ 返回科技板块在三大市场的覆盖情况
```

### 场景3：获取下周财报预览（全市场）

```
用户：获取下周财报日历
→ 搜索下周所有市场的财报
→ 筛选关注公司
→ 发送到飞书
```

### 场景4：按市场筛选

```
用户：下周美股有哪些财报？
→ 仅显示美股公司财报
```

### 场景5：按板块筛选

```
用户：科技板块下周有哪些财报？
→ 仅显示科技板块公司

用户：港股金融板块财报
→ 港股 + 金融板块双重筛选
```

### 场景6：查询特定公司（跨市场）

```
用户：NVDA什么时候发财报？
→ 自动识别为美股，返回时间和预期数据

用户：腾讯什么时候发财报？
→ 自动识别为港股00700.HK

用户：茅台财报时间？
→ 自动识别为A股600519
```

### 场景7：财报发布后自动分析

```
系统自动：
1. 财报发布后1小时触发
2. 搜索财报结果
3. 提取：营收、EPS、指引、业务亮点
4. 生成结构化摘要
5. 推送飞书
```

## CLI 命令

```bash
# 查看帮助
node index.js help

# 支持的股市
node index.js markets

# 查看板块
node index.js sectors          # 所有板块
node index.js sectors us       # 美股板块

# 获取财报预览
node index.js preview          # 所有市场
node index.js preview us       # 仅美股
node index.js preview hk tech  # 港股科技板块
node index.js preview cn finance # A股金融板块
node index.js sector tech      # 科技板块（跨市场）

# 查询公司
node index.js query NVDA       # 美股
node index.js query 00700.HK   # 港股
node index.js query 600519 cn  # A股

# 生成财报摘要
node index.js summary NVDA
node index.js summary 09988.HK

# 运行完整演示
node index.js demo
```

## 注意事项

1. **时间显示**：财报时间以各市场当地时间为准
   - 美股：美东时间（EST/EDT）
   - 港股：香港时间（HKT）
   - A股：北京时间（CST）

2. **盘前/盘后**：
   - 🌅 盘前：市场开盘前（美股 9:30 AM ET前）
   - 🌙 盘后：市场收盘后（美股 4:00 PM ET后）

3. **数据延迟**：数据仅供参考，不构成投资建议

## 故障排除

| 问题 | 解决 |
|------|------|
| 找不到股票 | 检查代码格式（港股加.HK，A股6位数字） |
| 板块无数据 | 该板块下周可能无财报安排 |
| 推送失败 | 检查飞书授权 |
